# 自適應個人視覺記憶系統語意檢索與自動化歸納架構
本專題預計將開發出一款本地應用程式，方便使用者在分類照片上享受如私人助理般的服務，使用者將相簿內的照片上傳後，即可自動化分類出如旅遊、美食及寵物等相簿，且具有可依時間、地點等建立出如: 2025日本、2026春季等相簿，好讓使用者方便找尋的功能，另外，此應用程式也將具備「以字搜圖」，只需打出關鍵字即可精準搜尋出照片的功用。

## Features
- 影像自動標記: 系統應能自動偵測上傳照片的內容(如:地點、物件、場景)，並進行基礎分類。
- 動態分類邏輯:支援「自適應」功能，系統能根據使用者行為或特定場景自動調整分類權重。
- 使用者自定義: 提供介面供使用者自行建立標籤或相簿並交由模型分類相片至該相簿內，亦可由使用者手動將相片移動至想要的相簿中。
- 多維度搜尋: 支援透過時間、地點等Metadata進行搜尋。
- 語義特徵搜尋: 利用Embedding技術，實現”A dog is playing a ball at the beach”這類描述性特徵的影像檢索。
- AI Agent相簿小幫手: 使用者可以透過對話框輸入指令與抽象複雜的語意進行搜尋以及根據照片資訊尋找相關資料。
